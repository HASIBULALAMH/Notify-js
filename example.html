Toastr.js Compatible Notification

Usage:

1. Include CSS and JS

<!-- Using CDN --> <link href="https://cdn.jsdelivr.net/gh/yourusername/notify/notify.min.css" rel="stylesheet"> <script src="https://cdn.jsdelivr.net/gh/yourusername/notify/notify.min.js"></script> <!-- OR download and include locally --> <link href="notify.min.css" rel="stylesheet"> <script src="notify.min.js"></script>

2. Show Notifications

// Basic usage toastr.success('Data saved successfully!'); toastr.error('Something went wrong!'); toastr.warning('Please check your input!'); toastr.info('This is an information message'); // With options toastr.success('Custom message', { duration: 5000, // 5 seconds position: 'bottom-right', closeButton: true }); // Configure defaults toastr.config({ position: 'top-right', duration: 3000, closeButton: true, preventDuplicates: true });

3. NPM Installation (optional)

npm install your-toastr-package // In your JavaScript file import toastr from 'your-toastr-package'; import 'your-toastr-package/dist/notify.min.css';
