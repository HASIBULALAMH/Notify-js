Notify.js

A lightweight, framework-agnostic toast notification library with zero dependencies.

Basic Usage

Show different types of notifications with a single line of code:

// Show a success notification
Notify.success('Operation completed successfully!');

// Show an error notification
Notify.error('Something went wrong!');

// Show a warning notification
Notify.warning('Please check your input!');

// Show an info notification
Notify.info('New message received');

Customization

Customize the appearance and behavior of notifications:

// Custom duration (10 seconds)
Notify.success('This will stay longer', { duration: 10000 });

// No auto-close
Notify.info('No auto-close', { duration: 0 });

// Change position
Notify.info('Bottom left position', { position: 'bottom-left' });

// Custom colors
Notify.info('Custom Color', { 
    colors: { 
        info: '#9b59b6',
        text: '#fff'
    } 
});

Global Configuration

Configure default settings for all notifications:

// Set default configuration
Notify.config({
    position: 'bottom-right',
    duration: 5000,
    preventDuplicates: true,
    maxToasts: 5,
    closeButton: true,
    icons: {
        success: '✓',
        error: '✕',
        warning: '!',
        info: 'i'
    },
    colors: {
        success: '#4CAF50',
        error: '#F44336',
        warning: '#FF9800',
        info: '#2196F3',
        text: '#FFFFFF',
        background: '#323232',
        close: '#FFFFFF'
    }
});

Advanced Usage

Handle notifications programmatically:

// Get a reference to a notification
const notification = Notify.info('Click to close', { 
    duration: 0 
});

// Close it programmatically
setTimeout(() => {
    notification.close();
}, 3000);

// Close all notifications
Notify.closeAll();
